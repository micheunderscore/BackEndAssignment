### Setup Step-by-Step

1. Clone the repo by pasting the command below in the opened cmd window

```
git clone https://github.com/micheunderscore/NotSafeForWork
```

2. Install project dependencies

```bash
$ bundle install
```

3. Migrate to the database

```bash
rails db:migrate
```

4. Run the server

```bash
$ rails server
```
